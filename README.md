# Chessigy

A basic chess application built with JavaFX.

## Prerequisites

- Java 17 or higher
- Maven 3.6+

## Building

```bash
mvn clean compile
```

## Running

```bash
mvn javafx:run
```

## Packaging

Packaging is a work-in-progress, an Arch Linux package can be built with: 
```bash
mvn clean install -Ppackage-arch
```
